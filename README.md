## How to run Wordpress + Woocommerce Locally with Docker
### how to run
#### step 1:
    create the directory: 'mysql_db_data', 'wordpress_data', 'plugins_data'
#### step 2:
    docker-compose up -d
#### step 3:
    hit the Wordpress url: http://localhost:8000/
#### step 4:
    install Wordpress
#### step 5:
    install Woocommerce
